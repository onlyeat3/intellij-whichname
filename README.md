# intellij-whichname(为取名) 变量命名工具，快速转换中文为变量名
翻译数据来源于互联网大厂对外公开的开发文档，未知词汇使用谷歌翻译
### 安装方法:
**在JetBrains系列产品中使用**
- <kbd>Preferences(Settings)</kbd> > <kbd>Plugins</kbd> > <kbd>Browse repositories...</kbd> > <kbd>find"whichname"</kbd> > <kbd>Install Plugin</kbd>
